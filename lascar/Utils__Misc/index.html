val iter_fix : ('a -> 'a -> bool) -> ('a -> 'a) -> 'a -> 'a

iter_fix eq f x iterates f, starting with x, until idempotence, i.e. computes f (f (f ... f(x)...)) until the condition f(x)=x is met. Equality test is performed with the eq argument
